# LUAScript

Loads and processes internal Lua scripts



## Enable module

To enable this module and and allow using the commands you need to ass `LUAScript = enabled` to the `[/modules]` section in nsclient.ini:

```
[/modules]
LUAScript = enabled
```

| Path / Section                        | Description |
